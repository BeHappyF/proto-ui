# ListView

## Examples
### Basic

``` html
<vi-list-view>
    <vi-item>Apple</vi-item>
    <vi-item>Banana</vi-item>
    <vi-item>Orange</vi-item>
</vi-list-view>
```

### `value` & `selected`

``` html
<vi-list-view>
    <vi-item>Apple</vi-item>
    <vi-item>Banana</vi-item>
    <vi-item selected>Orange</vi-item>
</vi-list-view>
```
