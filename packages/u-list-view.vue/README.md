# ListView

## Examples
### Basic

``` html
<u-list-view>
    <u-item>Apple</u-item>
    <u-item>Banana</u-item>
    <u-item>Orange</u-item>
</u-list-view>
```

### `value` & `selected`

``` html
<u-list-view>
    <u-item>Apple</u-item>
    <u-item>Banana</u-item>
    <u-item selected>Orange</u-item>
</u-list-view>
```
