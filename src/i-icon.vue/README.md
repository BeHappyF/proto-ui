# 图标 Icon

## 示例
### 基本形式

``` html
<i-icon name="arrow-up"></i-icon>
<i-icon name="arrow-down"></i-icon>
<i-icon name="arrow-left"></i-icon>
<i-icon name="arrow-right"></i-icon>
```

## API
### Props/Attrs

| Prop/Attr | Type | Default | Description |
| --------- | ---- | ------- | ----------- |
| name | String | | 图标名称 |
