# 水印 Watermark

``` html
<u-watermark text="Watermark"></u-watermark>
```

## API
### Props/Attrs

| Prop/Attr | Type | Default | Description |
| --------- | ---- | ------- | ----------- |
| text | String | `''` | 水印文本 |
| image | CanvasImageSource | | 水印图案。如果没有，则默认按文本绘制 |
| opacity | String | `0.1` | 透明度 |
